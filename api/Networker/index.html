Networker

A strictly-typed, action-based networking framework for Roblox. Handles Client-Server communication safely with built-in timeouts, retries, and strict type checking.

interface Network {
Namespace: string--

The unique identifier for this network channel.

Folder: Folder?--

The folder containing the RemoteEvent and RemoteFunction.

Event: RemoteEvent?--

Recipient​

</>
type Recipient = Player | {Player} | "all"

Defines the target(s) for a server-to-client push. Can be a single Player, an array of Players, or the string "all" to broadcast to everyone.

interface NetworkPayload {

NetworkOptions​

</>
interface NetworkOptions {
debug: boolean?--

If true, prints warning and success messages to the output for debugging.

timeout: number?--

The maximum time (in seconds) to wait for a response before cancelling.

retry: number?--

The number of times to retry a failed request.

usePromise: boolean?--

If true, the request will immediately return a Promise instead of yielding the thread.

}
